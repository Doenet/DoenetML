Set `maskLabel="true"` on a graphical component (or a stand-alone `<label>`) to give its label an opaque background so it stays legible when it overlaps an axis, grid line, or another object. Labels keep their transparent background by default. When masking is enabled, hovering a draggable object outlines its label as a cue that the object can be dragged.
