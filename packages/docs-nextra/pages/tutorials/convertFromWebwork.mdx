
# How to convert existing WeBWorK activities to DoenetML

If you're looking to convert WeBWorK problems to Doenet, consider using [the `webwork-to-doenetml` package in the DoenetML monorepo](https://github.com/Doenet/DoenetML/tree/main/packages/webwork-to-doenetml) to generate a first draft of the Doenet document. The script is very much a work in progress, so your mileage may vary.
