# ` <samplePrimeNumbers/>{:dn}`

`<samplePrimeNumbers/>{:dn}` is a [Math](../document_structure/essentialConcepts#component-types)
component that generates random prime numbers from a specified range of values.

## Attributes and Properties 
 <AttrDisplay name='samplePrimeNumbers'/> 
 <PropDisplay name='samplePrimeNumbers'/>

---

### Example: Default `<samplePrimeNumbers{:dn}`


```doenet-editor-horiz

<samplePrimeNumbers name="sample" />

<p>Prime selected = $sample</p>
```



Without any attributes specified, the default sampling range is $[2, 97]$. In this example, there are two ways to view different samples: selecting another variant, or refreshing the page.



---

### Example: Using a re-sampling button


```doenet-editor-horiz
<p><samplePrimeNumbers name="randomNumber"/></p>

<callAction 
  actionName="resample" 
  target="$randomNumber">
    <label>Get a New Prime</label>
</callAction>
```



Without any attributes specified, the default sampling range is $[2, 97]$. The ` <callAction>{:dn}` 
button can be used to resample the value.



---

### Attribute Example: numSamples


```doenet-example
<p>
  <samplePrimeNumbers numSamples="4" name="randomNumbers"/>
</p>

<callAction actionName="resample" target="$randomNumbers" >
  <label>Get New Primes</label>
</callAction>

<p>The second prime selected 
  <c>randomNumbers[2]</c> = $randomNumbers[2]
</p>
```



The ` <samplePrimeNumbers/>{:dn}` component is used with the `numSamples` attribute 
to select multiple random values in the default interval $[2,97]$.


---

### Attribute Example: minValue, maxValue


```doenet-example
<p>
  <samplePrimeNumbers 
    name="x" 
    minValue="5" 
    maxValue="25" 
    numSamples="5" 
  />
</p>

<callAction actionName="resample" target="$x">
  <label>Get New Primes</label>
</callAction>

<p>The second prime selected 
  <c>x[2]</c> = $x[2]
</p>
```



The ` <samplePrimeNumbers/>{:dn}` component is used with the `minValue` and the `maxValue` attributes 
to modify the default selection interval. Note that the interval endpoints provided do no need to 
be primes.



---

### Attribute Example: exclude


```doenet-example
<p>
  <samplePrimeNumbers 
    name="x" 
    minValue="2" 
    maxValue="10" 
    numSamples="10" 
    exclude="3"
  />
</p>

<callAction actionName="resample" target="$x">
  <label>New Primes</label>
</callAction>

<p>The second prime selected 
  <c>x[2]</c> = $x[2]
</p>
```



Values can be excluded from the sampled range using the `exclude` attribute.
