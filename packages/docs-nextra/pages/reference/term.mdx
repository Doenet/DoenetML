# `<term>{:dn}`

<ComponentDisplay name='term'/>

`<term>{:dn}` is a [Paragraph Markup](../concepts/essentialConcepts#component-types) 
component that renders the enclosed text in the emphasized 
styling associated with a defined vocabulary word.

<AttrPropDisplay name='term'/>

## Examples

### Example: Using `<term>{:dn}` to highlight important words


```doenet-editor-horiz
<p>What does <term>Doenet</term> stand for? 
  <term>Doenet</term>
means the <sq>Distributed Open Education Network</sq>.</p>
```



In the example above, `<term>{:dn}` is used when referring to the word "Doenet", which is associated with a definition.
