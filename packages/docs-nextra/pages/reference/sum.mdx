# ` <sum>{:dn}`


` <sum>{:dn}` is a [Math Operator](../document_structure/essentialConcepts/#componentTypes) 
component that renders the sum of its arguments.


## Attributes and Properties 
 <AttrDisplay name='sum'/> 
 <PropDisplay name='sum'/>
 

---

### Example: ` <sum>{:dn}` of a nested list of numbers


```doenet-editor-horiz
<sum>0 2.5 1 4.1 3</sum>
```



` <sum>{:dn}` renders the sum of values in a list of numbers separated by spaces.



---

### Example: ` <sum>{:dn}` of a named list


```doenet-editor-horiz
<numberList name="numList">1.2 9.9 17.3</numberList>

<p>Sum = <sum>$numList</sum></p> 
```



` <sum>{:dn}` renders the sum of values in a named ` <numberList>{:dn}`.



---

### Example: ` <sum>{:dn}` of a randomly selected set of numbers


```doenet-editor-horiz

<selectRandomNumbers 
  numToSelect="7" 
  from="-500" 
  to="500" 
  type="discreteUniform" 
  name="randomNumberList"
/>

<p>The list of random numbers selected is: 
  $randomNumberList</p>
<p>Sum of selected numbers: <sum>$randomNumberList</sum></p>
```



` <sum>{:dn}` renders the sum of values in a list of numbers that are generated with 
the ` <selectRandomNumbers/>{:dn}` component. 



---

### Example: ` <sum>{:dn}` of symbolic maths


```doenet-editor-horiz
<mathList name="mList">
  x 3y 2x^2 y -3x x^2
</mathList>

<p>Sum = 
  <sum name="sumOfMaths">$mList</sum>
</p> 
<p>Simplified = 
  <sum extend="$sumOfMaths" simplify/>
</p> 
```



` <sum>{:dn}` renders the sum of values in a named ` <mathList>{:dn}`. Since ` <sum>{:dn}` is a math component, ` <math>{:dn}` attributes like `simplify` are also available.
