# ` <isInteger>{:dn}`


` <isInteger>{:dn}` is a [Logic](../document_structure/essentialConcepts#component-types) 
component that returns the boolean value of `true` or `false` depending on whether the value enclosed is an integer.

### Example: Basic use


```doenet-editor-horiz
<p>Enter an integer (or not): <mathInput name="mi" prefill="4/2"/></p>
<p>This value is an integer: </p>
<isInteger>$mi</isInteger>
```



Tne ` <isInteger>{:dn}` component checks to see whether the enclosed value is an integer.
