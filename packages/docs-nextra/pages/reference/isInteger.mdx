# `<isInteger>{:dn}`

<ComponentDisplay name='isInteger'/>

`<isInteger>{:dn}` is a [Logic](../document_structure/essentialConcepts#component-types) 
component that returns the boolean value of `true` or `false` depending on whether the value enclosed is an integer.

<AttrPropDisplay name='isInteger'/>

## Examples

### Example: Basic use


```doenet-editor-horiz
<p><mathInput name="mi" prefill="4/2"><label>Enter an integer (or not)</label></mathInput></p>
<p>This value is an integer: </p>
<isInteger>$mi</isInteger>
```



The `<isInteger>{:dn}` component checks to see whether the enclosed value is an integer.
