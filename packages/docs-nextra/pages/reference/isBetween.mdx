# ` <isBetween>{:dn}`

` <isBetween>{:dn}` is a [Logic](../document_structure/essentialConcepts#component-types) component
that returns the boolean value of `true` or `false` depending on whether a value is enclosed in a specified interval.

## Attributes and Properties 
 <AttrDisplay name='isBetween'/> 
 <PropDisplay name='isBetween'/>
 
---

### Example: Basic use


```doenet-editor-horiz
<p>Enter a number: <mathInput name="mi" prefill="0"/></p>
<p>Your number is between <m>0</m> and <m>5</m>: </p>
<isBetween limits="0 5" >$mi</isBetween>
```



The endpoints of the desired interval are entered as a list of numbers in the `limits` attribute. 
By default, endpoints are included in the interval.



---

### Attribute Example: strict


```doenet-editor-horiz
<p>Enter a number: <mathInput name="mi" prefill="0"/></p>
<p>Your number is between <m>0</m> and <m>5</m>: </p>
<isBetween limits="0 5" strict>$mi</isBetween>
```



The `strict` attribute is included to exclude the endpoints of the interval.
