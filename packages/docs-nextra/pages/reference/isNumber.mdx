# ` <isNumber>{:dn}`

` <isNumber>{:dn}` returns the boolean value of `true` or `false` depending on whether a value enclosed is a number.

## Attributes and Properties 
 <AttrDisplay name='isNumber'/> 
 <PropDisplay name='isNumber'/>

---

### Example: basic use


```doenet-editor-horiz
<p>Enter a number (or not): <mathInput name="mi" prefill="0"/></p>
<p>This value is a number: </p>
<isNumber>$mi</isNumber>
```



Tne ` <isNumber>{:dn}` component checks to see whether the enclosed value is a number.
