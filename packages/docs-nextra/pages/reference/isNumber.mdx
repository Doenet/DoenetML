# `<isNumber>{:dn}`

<ComponentDisplay name='isNumber'/>

`<isNumber>{:dn}` is a [Logic](../concepts/essentialConcepts#component-types)
component that returns the boolean value of `true` or `false` depending on whether the value enclosed is a real number.

<AttrPropDisplay name='isNumber'/>

## Examples

### Example: Basic use


```doenet-editor-horiz
<p><mathInput name="mi" prefill="0"><label>Enter a number (or not)</label></mathInput></p>
<p>This value is a real number: </p>
<isNumber>$mi</isNumber>

<p><mathInput name="ci" prefill="sqrt(-1) + 1" displaySmallAsZero><label>Enter a complex number</label></mathInput></p>
<p>You have entered the complex number: 
<number name="n2">$ci</number></p>

<p>This value is a real number: </p>
<isNumber>$n2</isNumber>
```



The `<isNumber>{:dn}` component checks to see whether the enclosed value is a real number.
