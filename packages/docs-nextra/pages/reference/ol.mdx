# ` <ol>{:dn}`

` <ol>{:dn}` is a [Sectional](../document_structure/essentialConcepts#component-types)
component that defines an ordered (numbered) list.  List items are nested within using an ` <li>{:dn}` tag.

### Example: ` <ol>{:dn}` with nested ` <li>{:dn}`


```doenet-editor-horiz
<p>Flavors</p>
<ol>
  <li>Chocolate</li>
  <li>Vanilla</li>
  <li>Strawberry</li>
</ol>

<p>Shapes</p>
<ol>
  <li>
    <p>
      <graph size="tiny" displayMode="inline" displayXaxis="false" displayYaxis="false" fixAxes
        description="A graph of a circle"
      >
        <circle radius="5"/>
      </graph>
    </p>
  </li>
  <li>
    <p>
      <graph size="tiny" displayMode="inline" displayXaxis="false" displayYaxis="false" fixAxes
        description="A graph of a pentagon"
      >
        <regularPolygon radius="5"/>
      </graph>
    </p>
  </li>
  <li>
    <p>
      <graph size="tiny" displayMode="inline" displayXaxis="false" displayYaxis="false" fixAxes
        description="A graph of a rectangle"
      >
        <rectangle height="8" width="8"/>
      </graph>
    </p>
  </li>
</ol>

<p>Equations</p>
<ol>
  <li><p><m>A = \pi r^2</m></p></li>
  <li><p><m>F = ma</m></p></li>
  <li><p><m>E = mc^2</m></p></li>
</ol>
```



` <ol>{:dn}` renders an ordered list, with list items given by ` <li>{:dn}` children. ` <ol>{:dn}` 
should not contain any components other than ` <li>{:dn}` children. Individual ` <li>{:dn}` children 
may themselves contain graphs, problems, paragraphs, and other content.



---

### Attribute Example: level


```doenet-editor-horiz
<ol>
  <li>Animals
    <ol level="2">
      <li>dog</li>
      <li>zebra</li>
      <li>whale
        <ol level="3">
          <li>blue whale</li>
          <li>beluga whale</li>
          <li>killer whale</li>
        </ol>
      </li>
    </ol>
  </li>
  <li>Plants
    <ol level="2">
      <li>oak tree</li>
      <li>rose</li>
      <li>eucalyptus</li>
    </ol> 
  </li>
</ol>
```



When nesting ` <ol>{:dn}` components, the `level` attribute can be used to specify a 
change in the counting marker for the inner lists.



---

### Attribute Example: marker


```doenet-editor-horiz
<ol marker="A">
  <li>red</li>
  <li>blue</li>
  <li>green</li>
</ol>
<ol marker="1">
  <li>red</li>
  <li>blue</li>
  <li>green</li>
</ol>
<ol marker="I">
  <li>red</li>
  <li>blue</li>
  <li>green</li>
</ol>
<ol marker="a">
  <li>red</li>
  <li>blue</li>
  <li>green</li>
</ol>
```



Use the `marker` attribute to indicate a specific counting marker for the list. See 
the `level` example for the default markers at each level.
