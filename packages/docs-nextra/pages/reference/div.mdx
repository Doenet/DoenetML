# ` <div>{:dn}`

` <div>{:dn}` groups doenetML content into an unformatted container that can be named.

## Attributes and Properties 
 <AttrDisplay name='div'/> 
 <PropDisplay name='div'/>


---

### Example: ` <div>{:dn}` as a general container element


```doenet-editor-horiz
<booleanInput name="hideContent">
  <label>Hide the <tag>div</tag></label>
</booleanInput>
<div name="contentDiv" hide="$hideContent">
  <p><lorem generateWords="10"/></p>
  <graph size="tiny"/>
  <slider/>
</div>
<booleanInput name="copyContent">
  <label>Copy the <tag>div</tag></label>
</booleanInput>
<conditionalContent condition="$copyContent=true">
  $contentDiv
</conditionalContent>
```



A ` <div>{:dn}` can be used to group portions of content that are to be collectively hidden or copied.
