# ` <convertSetToList>{:dn}`


` <convertSetToList>{:dn}`<idx>` <convertSetToList>{:dn}`</idx> takes as an argument a set of values that is 
generated by a set-generating component such as <ref uri="doenet:doenetID=_HtetAL1AhP13dJyv4sP2H">` <selectFromSequence/>{:dn}`</ref>, 
<ref uri="doenet:doenetId=_HFhH7id_K4u5E1OiZTTby">` <selectRandomNumbers/>{:dn}`</ref> or ` <sampleRandomNumbers/>{:dn}` and renders 
it as a comma-separated list.

## Attributes and Properties 
 <AttrDisplay name='convertSetToList'/> 
 <PropDisplay name='convertSetToList'/>

---

### Example: Convert a <tage>selectFromSequence</tage> to a list


```doenet-editor-horiz
<p>
  <selectFromSequence from="-10" to="10" 
     numberToSelect="3" name="set"/>
</p>
<p>
  <convertSetToList>$set</convertSetToList>
</p>
```



The ` <convertSetToList>{:dn}` component is used to render the set of values generated by the <ref uri="doenet:doenetID=_HtetAL1AhP13dJyv4sP2H">` <selectFromSequence/>{:dn}`</ref> component as a comma-separated list.



---

### Example: Convert a <tage>selectRandomNumbers</tage> to a list


```doenet-editor-horiz
<p>
  <selectRandomNumbers mean="40" numberToSelect="5" type="gaussian" name="set"/>
</p>
<p>
  <convertSetToList>$set</convertSetToList>
</p>
```



The ` <convertSetToList>{:dn}` component is used to render the set of values generated by the <ref uri="doenet:doenetId=_HFhH7id_K4u5E1OiZTTby">` <selectRandomNumbers/>{:dn}`</ref> component as a comma-separated list.



---

### Example: Convert a <tage>sequence</tage> to a list


```doenet-editor-horiz
<p>State the meaning of life as the sum of an arithmetic sequence with an increment of 3:</p>
<p><sequence from="-3" to="15" step="3" name="byThrees"/></p>
<p>
  <convertSetToList name="pretty">$byThrees</convertSetToList>
</p>
<p>The meaning of life is <m>\sum{$pretty} = <sum>$byThrees</sum></m>
</p>
```



The ` <convertSetToList>{:dn}` component is used to render the set of values generated by the <ref uri="doenet:doenetId=_HFhH7id_K4u5E1OiZTTby">` <sequence/>{:dn}`</ref> component as a comma-separated list. Note that, for computational purposes (such as within a [` <sum>{:dn}`](sum.mdx)), the original ` <sequence/>{:dn}` should be used.
