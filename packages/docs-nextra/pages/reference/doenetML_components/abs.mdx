# `<abs>{:dn}`

## What it does
The `<abs>{:dn}` component renders the absolute value of the enclosed math.

## Attributes and Properties
The `<abs>{:dn}` component is a math operator, and shares relevant attributes and properties with the `<math>{:dn}` component.

## Examples
<a id="math-ex"></a>
### Example: Render the absolute value of a math expression

The `<abs>{:dn}` component is used to render the absolute value of a math expression.
```doenet-editor-horiz
<m>|5 - 8| = <abs>5 - 8</abs></m>
```


---

<a id="mathInput-ex"></a>
### Example: Absolute value of `<mathInput>`

The `<abs>{:dn}` component is used to render the absolute value of a named `<mathInput/>{:dn}`.
```doenet-editor-horiz
    <p>Type any number: <mathInput name="userN"/></p>
    <p><m>|$userN| = <abs>$userN</abs></m></p>
```


---

<a id="numProps-ex"></a>
### Example: Absolute value of numerical properties

Children of the `<abs>{:dn}` component can include references to `<mathInput>{:dn}` values or other `math` quantities, such as single-valued numerical properties of graphical objects. Invalid children are strings and tuples.
```doenet-editor-horiz
<graph>
    <point name="dp1">(4,4)</point>
    <line name="l1" through="$dp1 (0,0)"/>
</graph>
<p>The slope is <m>m = $l1.slope</m></p>
<p>The absolute value of the slope is <m>|m| = <abs>$l1.slope</abs></m></p>
```



<a id="function-ex"></a>
### Example: Absolute value of a function

The `<abs>{:dn}` component is used to render the absolute value of a named `<function>{:dn}` within a `<graph>{:dn}`.
```doenet-editor-horiz
<graph size="small" name="g2">
    <function name="f">(1/4)x^2 - 4</function>
    <function styleNumber="2">
        <abs>$f</abs>
    </function>
</graph>
```


---
### Special usage notes

Note that the `<abs>{:dn}` component cannot be used directly in attribute specification. You can, however, use the `|` notation within an attribute specification. 
You can also define a `<math>{:dn}` component separately, which contains the `<abs>{:dn}` component, and then reference it in an attribute specification.
