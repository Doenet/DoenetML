# Index by Component Type

## Paragraph markup components
These components can be placed within a `<p>` tag to change the rendered appearance of the enclosed text. (The exception is `<nbsp/>`, which is a self-closing tag that renders a blank space.)
| Component                                                              | Description                                                                           |
| ---------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| [`<alert>`](alert.mdx)                                                 | bold font                                                                             |
| [`<em>`](em.mdx)                                                       | italicised font                                                                       |
| [`<term>`](term.mdx)                                                   | a vocabulary word                                                                     |
| [`<footnote>`](footnote.mdx)                                           | creates an expandable, in-place footnote                                              |
| [`<pre>`](pre.mdx)                                                     | preserves verbatim text; renders text as input into the editor                        |
| [`<nbsp/>`](nbsp.mdx)                                                  | forced white space                                                                    |
| [`<q>`](q.mdx)                                                         | quotation                                                                             |
| [`<c>`](https://www.doenet.org/portfolioviewer/_AtGGf77DIkIt1-XUDYQSs) | renders default font for a code example                                               |
| [`<attr>`](attr.mdx)                                                   | renders default font for an attribute                                                 |
| [`<m>`](m.mdx)                                                         | inline mathematical display text                                                      |
| [`<me>`](me.mdx)                                                       | centered mathematical display text                                                    |
| [`<men>`](men.mdx)                                                     | centered, numbered mathematical display text                                          |
| [`<md>`](md.mdx)                                                       | block component for centered, aligned equations; container for `<mrow>`               |
| [`<mdn>`](mdn.mdx)                                                     | block component for centered, numbered, and aligned equations; container for `<mrow>` |
| [`<mrow>`](mrow.mdx)                                                   | defines individual row content when nested within an `<md>` or `<mdn>`                |
