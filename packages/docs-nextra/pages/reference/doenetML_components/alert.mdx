# `<alert>`

## What it does
`<alert>` is a [**Paragraph Markup**](../componentClasses) component which renders the enclosed text in bold font.

## Attributes and Properties
The `<alert>` component has no attributes or properties.


### Example: Basic use
```html
<p>Take the <alert>right</alert> turn.</p>
```
