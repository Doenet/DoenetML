# Component Classes

## Paragraph markup components
These components can be placed within a `<p>` tag to change the rendered appearance of the enclosed text. (The exception is `<nbsp/>`, which is a self-closing tag that renders a blank space.)
* `<alert>` bold font
* `<em>`  italicised font
* `<term>` a vocabulary word, renders in bold font
* `<footnote>` creates an expandable, in-place footnote
* `<pre>`
* `<nbsp/>`
* `<q>`
* `<c>`
* `<attr>`
* `<m>`
* `<me>`
* `<men>`
* `<md>`
* `<mdn>`
