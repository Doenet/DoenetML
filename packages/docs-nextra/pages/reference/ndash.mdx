# `<ndash/>{:dn}`

<ComponentDisplay name='ndash'/>

`<ndash/>{:dn}` is a [Paragraph Markup](../concepts/essentialConcepts#component-types)
component used to insert an en-dash within a paragraph.


<AttrPropDisplay name='ndash'/>


## Examples

### Example: `<ndash/>{:dn}` in a paragraph


```doenet-example
<p>See pages 12<ndash/>15 for the discussion of inverse functions.</p>
```



An en-dash is typically used to indicate a range of values, such as a span of page numbers.
