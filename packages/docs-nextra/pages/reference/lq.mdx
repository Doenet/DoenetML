# `<lq/>{:dn}`

<ComponentDisplay name='lq'/>

`<lq/>{:dn}` is a [Paragraph Markup](../concepts/essentialConcepts#component-types)
component.


<AttrPropDisplay name='lq'/>


## Examples

### Example: `<lq/>{:dn}` in a paragraph


```doenet-example
<p>The instructions began, <lq/>Solve the following equation<rq/>.</p>
```



For most cases, the [`<q>{:dn}`](q) component is a more convenient way to wrap text in
matching curly quotation marks. Use `<lq/>{:dn}` and [`<rq/>{:dn}`](rq) when you need to
place each mark independently.
