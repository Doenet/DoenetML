# `<tag>{:dn}`, ` <tage/>{:dn}`

` <tag>{:dn}` is a [Paragraph Markup](../document_structure/essentialConcepts/#componentTypes) 
component which renders the enclosed text in the default font for component tags.   

Similarly, a `<tage/>{:dn}` component may be used to render 
a self-closing tag (i.e. `< componentName />`).  Use of ` <tag>{:dn}` and ` <tage/>{:dn}` will prevent errors with 
rendering these symbols, allowing them to be used for display rather than as functional components within the document.

## Attributes and Properties 
 
The ` <tag>{:dn}` and ` <tage/>{:dn}` components do not have any attributes or properties.



### Example: Use ` <tag>{:dn}` for correct rendering 


```doenet-editor-horiz
<p><alert>Example With the Tag Component (Correctly Displayed):</alert></p>

<p>When creating a new webpage, you can use the <tag>p</tag> 
  to generate a block of text with HTML.  To insert a photo you should
  use the <tage>image</tage> tag.
</p>
```

The author's intention is to describe the use of the ` <p>{:dn}` and ` <image\>{:dn}` tags in writing HTML. The ` <tag>{:dn}` component is 
therefore used to render the opening and closing brackets for the tags correctly.
