# `<round>{:dn}`

<ComponentDisplay name='round'/>

`<round>{:dn}` is a [Math Operator](../concepts/essentialConcepts#component-types)
component that renders a numerical value rounded to the user-specified number of decimal places or digits.  Default rounding is to the nearest integer.

<AttrPropDisplay name='round'/>

## Examples

### Example: Default rounding


```doenet-editor-horiz
<p><mathInput name="n"><label>Enter a number to be rounded</label></mathInput></p>
<p>Rounded number = <round>$n</round></p>
```



Default rounding is to the nearest integer.



---

### Example: numDecimals


```doenet-editor-horiz
<p>
  <mathInput name="n"><label>Enter a number with more than 2 decimal places</label></mathInput></p>
<p>Rounded number = 
  <round numDecimals="2">$n</round>
</p>  
```



The `numDecimals` attribute specifies how many decimal places to round to.



---

### Example: numDigits


```doenet-editor-horiz
<p>
  <mathInput name="n"><label>Enter a number with more than 2 significant figures</label></mathInput></p>
<p>Rounded number = 
  <round numDigits="2">$n</round>
</p>
```



The `numDigits` attribute specifies how many significant figures to round to.
