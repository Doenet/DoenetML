# `<mdash/>{:dn}`


`<mdash/>{:dn}` is a [Paragraph Markup](../document_structure/essentialConcepts#component-types)
component that renders an em-dash (—) character.


<AttrPropDisplay name='mdash'/>


---

### Example: `<mdash/>{:dn}` in a paragraph


```doenet-example
<p>The answer<mdash/>after all of that work<mdash/>was surprisingly simple.</p>
```



An em-dash is typically used to set off a parenthetical phrase or to mark an abrupt change in thought.
