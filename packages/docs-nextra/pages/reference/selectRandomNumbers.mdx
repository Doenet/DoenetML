# `<selectRandomNumbers/>{:dn}`

` <selectRandomNumbers/>{:dn}` is a [Math](../document_structure/essentialConcepts#component-types)
component that generates document variants by selecting random numbers from a specified range of 
values or within a normal distribution around a specified mean.

## Attributes and Properties 
 <AttrDisplay name='selectRandomNumbers'/> 
 <PropDisplay name='selectRandomNumbers'/>

---

### Example: Default `<selectRandomNumbers/>{:dn}`


```doenet-editor-horiz
<selectRandomNumbers name="s" />

<p>Number selected = $s</p>
```



Without any attributes specified, the default selection  `type` is `uniform`, and the default interval is $[0, 1)$.

Select a new page variant from the pulldown menu at the top of the 
editor to see different variants of the document.

---

### Attribute Example: numToSelect


```doenet-editor-horiz
<selectRandomNumbers
  name="s"
  numToSelect="4"
/>

<p>The third selection is <c>s[3]</c> = $s[3]</p>
```



The ` <selectRandomNumbers/>{:dn}` component is used with the `numToSelect` attribute to select multiple 
random values in the default interval $[0,1)$. The default `type` is `uniform`. 

Select a new page variant from the pulldown menu at the top of the 
editor to see different variants of the document.

---

### Example: Assign descriptive names to selections


```doenet-editor-horiz
<setup>
<selectRandomNumbers name="s" numToSelect="3"/>
<number name="a" extend="$s[1]"/>
<number name="b" extend="$s[2]"/>
<number name="c" extend="$s[3]"/>
</setup>

<p>a = $a</p>
<p>b = $b</p>
<p>c = $c</p>
```

Three numbers are selected. 
Selections can be assigned more descriptive names by defining individual components
which reference the selected values.

Select a new page variant from the pulldown menu at the top of the 
editor to see different variants of the document.

### Attribute Example: from, to


```doenet-editor-horiz
<selectRandomNumbers
  name="x" 
  from="-5" 
  to="5" 
  numToSelect="5"
/>

<p>The third selection is <c>x[3]</c> = $x[3]</p>
```



The ` <selectRandomNumbers/>{:dn}` component is used with the `from` and the `to` attributes to modify the 
default selection interval. The default `type` is `uniform`.


Select a new page variant from the pulldown menu at the top of the 
editor to see different variants of the document.



---

### Attribute Example: type = "discreteUniform"


```doenet-editor-horiz
<selectRandomNumbers
  name="x" 
  type="discreteUniform"
  from="0" 
  to="10" 
  numToSelect="5"
/>

<p>The third selection is <c>x[3]</c> = $x[3]</p>
```



The `type` attribute of the ` <selectRandomNumbers/>{:dn}` component is specified as `discreteUniform`. 
The `discreteUniform` selection generates values from an evenly spaced range defined by the 
attributes `to`, `from`, and `step`. If left unspecified, the default `step` is 1.


Select a new page variant from the pulldown menu at the top of the 
editor to see different variants of the document.

### Attribute Example: exclude


```doenet-editor-horiz
<selectRandomNumbers 
  name="x" 
  type="discreteUniform" 
  from="-4" 
  to="4" 
  numToSelect="4" 
  exclude="0 1 -1"
/>

<p>
  <math name="m">
    $x[1]/$x[2] + $x[3]/$x[4]

  <math 
    extend="$m" 
    simplify="numberspreserveorder"/>

```


The `type` attribute of the ` <selectRandomNumbers/>{:dn}` component is specified 
as `discreteUniform`. Values can be excluded from the specified range used with this type using the `exclude` attribute.

editor to see different variants of the document.

### Attribute Example: type = "gaussian"


```doenet-editor-horiz 
<selectRandomNumbers 
  name="x" 
  type="gaussian" 
  numToSelect="10"
  displayDigits="4"
/>

<p>The third selection is <c>x[3]</c> = $x[3]</p>  
```



The `type` attribute of the ` <selectRandomNumbers/>{:dn}` component is 
specified as `gaussian`. The `gaussian` selection generates standard normal 
variables (mean 0, variance 1). Alternatively, the `mean` as well as the `variance` 
or the `standardDeviation` attributes may be defined.


Select a new page variant from the pulldown menu at the top of the 
editor to see different variants of the document.
