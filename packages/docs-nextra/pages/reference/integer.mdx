# `<integer>{:dn}`

<ComponentDisplay name='integer'/>

`<integer>{:dn}` is a [Math](../document_structure/essentialConcepts#component-types) 
component that rounds its content to the nearest integer.

<AttrPropDisplay name='integer'/>

## Examples

### Example: `<integer>{:dn}` of nested number


```doenet-editor-horiz
<integer name="int">2.3</integer>

<p>$int </p>
```



The named `<integer>{:dn}` stores the enclosed number as the nearest integer value.



---

### Example: `<integer>{:dn}` of user-input


```doenet-example
<p><mathInput name="userN"><label>Enter a number with a decimal or fractional part</label></mathInput></p>

<p>The nearest integer is: <integer>$userN</integer></p>
```



The named `<mathInput/>{:dn}` is stored by the `<integer>{:dn}` component as 
the nearest integer value.
