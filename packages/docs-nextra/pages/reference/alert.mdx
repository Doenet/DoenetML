# `<alert>{:dn}`

<ComponentDisplay name='alert'/>

`<alert>{:dn}` is a [Paragraph Markup](../document_structure/essentialConcepts#component-types) component which 
renders the enclosed text in bold font.

<AttrPropDisplay name='alert'/>

## Examples

### Example: Render the enclosed text in bold font


```doenet-example
<p>Take the <alert>right</alert> turn.</p>
```



The `<alert>{:dn}` expression is used to render the enclosed text in bold font.
