# ` <tabular>{:dn}`


` <tabular>{:dn}` is a [Sectional](../document_structure/essentialConcepts/#componentTypes)
component that contains the data and style specifications for a [` <table>{:dn}`](table.mdx). Nested 
inside a ` <tabular>{:dn}` component are the [` <row>{:dn}`](row.mdx) and [` <cell>{:dn}`](cell.mdx) 
components. 


## Attributes and Properties

 <PropDisplay name='tabular'/>

---

### Example: Default ` <tabular>{:dn}` without attributes


```doenet-editor-horiz
<tabular>
  <row>
    <cell>1</cell>
    <cell>2</cell>
    <cell>3</cell>
  </row>
  <row>
    <cell>a</cell>
    <cell>b</cell>
    <cell>c</cell>
  </row>
</tabular>
```



A ` <tabular>{:dn}` can be used alone, without attributes, simply to organize content in a regular structure.

Borders and dividing lines within a ` <table>{:dn}` can be specified within the ` <tabular>{:dn}` component using the attributes: `top`, `bottom`, `left`, and `right`. If subsequent ` <row>{:dn}` or ` <cell>{:dn}` components also have the same attribute specified, the attribute corresponding to the lowest hierarchical level takes precedence.  All border/dividing line attributes accept either `major`, `medium`, `minor`, or `none` (default).

```doenet-editor-horiz

      <cell>b</cell>
```

The `height` and `vAlign` attributes within a ` <tabular>{:dn}` can control the height and vertical alignment of individual rows for the entire table.

### Attribute Example: width/height


```doenet-editor-horiz
<table>
  <tabular top="minor" bottom="minor" left="minor" right="minor" width="100%" height="200">
    <row>
      <cell>ðŸŸ£</cell>
      <cell>ðŸ”µ</cell>
      <cell>ðŸ”´</cell>
    </row>
    <row>
      <cell>ðŸ˜†</cell>
      <cell>ðŸ¥²</cell>
      <cell>ðŸ˜ </cell>
    </row>
  </tabular>
</table>
```



The `width` and `height` attributes for the ` <tabular>{:dn}` component accept measurements for the extents of the ` <tabular>{:dn}`. Width is specified as a percentage of screen width, and height is specified in absolute measurements. Individual rows and cells are then scaled in proportion to these extents.



---

### Attribute Example: hAlign


```doenet-editor-horiz
<table>
  <tabular left="minor" right="minor" top="minor" bottom="minor" width="100%" hAlign="right">
    <row>
      <cell>1</cell>
      <cell>2</cell>
      <cell>3</cell>
    </row>
    <row>
      <cell>a</cell>
      <cell>b</cell>
      <cell>c</cell>
    </row>
  </tabular>
</table>
```



The `hAlign` attribute within a ` <tabular>{:dn}` controls the horizontal alignment of individual rows for the entire table. Valid values are `left` (default), `center`, `right`, and `justify`.

```doenet-editor-horiz
<p><c>top</c> = $tabular1.top</p>
<p><c>left</c> = $tabular1.left</p>

<p><c>right</c> = $tabular1.right</p>
```

The above-listed attributes of the ` <tabular>{:dn}` component are also accessible as properties.
