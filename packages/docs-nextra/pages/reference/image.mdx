# ` <image/>{:dn}`

` <image/>{:dn}`renders an image from a hyperlink or an uploaded file.

## Attributes and Properties 
 <AttrDisplay name='image'/> 
 <PropDisplay name='image'/>





---

### Example: <tage>image</tage> from a hyperlink


```doenet-editor-horiz
<image source="https://upload.wikimedia.org/wikipedia/commons/1/16/DoricParthenon.jpg" description="A corner of a Doric temple with column capital, architrave, and triglyph"/>
```



An ` <image/>{:dn}` may be inserted from a hyperlink using the `source` attribute. The `description` attribute is required for accessibility.



---

### Example: <tage>image</tage> in a ` <figure>{:dn}`


```doenet-editor-horiz
<figure>
  <image size="full" source="https://upload.wikimedia.org/wikipedia/commons/thumb/6/67/1684_untitled_engraving_of_Sea_Monsters_Attacking_A_Sailing_Vessel.jpg/640px-1684_untitled_engraving_of_Sea_Monsters_Attacking_A_Sailing_Vessel.jpg" description="An engraving of sea monsters attacking a sailing vessel"/>
  <caption>Engraving of sea monsters attacking a vessel.</caption>
</figure>
```



An ` <image/>{:dn}` is often nested within a ` <figure>{:dn}` along with a ` <caption>{:dn}`. The `description` attribute is required for accessibility.

### Example: <tage>image</tage> on a ` <graph>{:dn}`


```doenet-editor-horiz
<graph size="small">
  <image source='/media/bafkreifp2vbffgblocyvziejmp7hihzsx4dfkz6x4fayarcnfbb27pygaa.png' description='cartoon squirrel' asfilename='squirrel-clipart-md.png' mimeType='image/png' size="small" anchor="(1,2)"/>
</graph>
```



An ` <image/>{:dn}` can be placed on a ` <graph>{:dn}`. Control the size of the image with the `size` attribute and its initial position with the `anchor` attribute.



---

### Example: width


```doenet-editor-horiz
<image width="100px" name="doric" source="https://upload.wikimedia.org/wikipedia/commons/1/16/DoricParthenon.jpg" description="A corner of a Doric temple with column capital, architrave, and triglyph"/>
```



The `width` attribute specifies the absolute width of the image. Doenet determines the `size` category corresponds to that width and sizes the image accordingly.



---

### Example: size


```doenet-editor-horiz
<image size="tiny" name="doric" source="https://upload.wikimedia.org/wikipedia/commons/1/16/DoricParthenon.jpg" description="A corner of a Doric temple with column capital, architrave, and triglyph"/>
<image copySource="doric" size="small"/>
<image copySource="doric" size="full"/>
```



The `size` attribute specifies one of 5 size categories for the ` <image/>{:dn}`: `tiny`, `small`, `medium`, `large`, or `full`.



---

### Example: aspectRatio


```doenet-editor-horiz
<image size="small" aspectRatio="2/1" name="doric" source="https://upload.wikimedia.org/wikipedia/commons/1/16/DoricParthenon.jpg" description="A corner of a Doric temple with column capital, architrave, and triglyph"/>
```



The `aspectRatio` attribute allows for customization of the image's default proportions.



---

### Example: displayMode


```doenet-editor-horiz
<image size="tiny" displayMode="inline" name="doric" source="https://upload.wikimedia.org/wikipedia/commons/1/16/DoricParthenon.jpg" description="A corner of a Doric temple with column capital, architrave, and triglyph"/>
<image copySource="doric" displayMode="block"/>  
```



The `displayMode` attribute affects how the ` <image/>{:dn}` is positioned on screen. The default mode is "block", which renders the image centered on screen. The available modes are `inline`, and `block`.



---

### Example: horizontalAlign


```doenet-editor-horiz
<image size="tiny" horizontalAlign="right" name="doric" source="https://upload.wikimedia.org/wikipedia/commons/1/16/DoricParthenon.jpg" description="A corner of a Doric temple with column capital, architrave, and triglyph"/>
```



The `horizontalAlign` attribute specifies the horizontal position on-screen for the default block-mode style image. Valid values are `left`, `center`, and `right`, with a default of `center`.

### Example: source


```doenet-editor-horiz
<image size="small" source="https://upload.wikimedia.org/wikipedia/commons/thumb/6/67/1684_untitled_engraving_of_Sea_Monsters_Attacking_A_Sailing_Vessel.jpg/640px-1684_untitled_engraving_of_Sea_Monsters_Attacking_A_Sailing_Vessel.jpg" description="An engraving of sea monsters attacking a sailing vessel"/>
```



The `source` attribute specifies the url for a hyperlinked ` <image/>{:dn}` and the filepath for an uploaded ` <image/>{:dn}`. For uploaded images, this attribute will be automatically populated with the correct data when the "Copy Code" button is clicked.

### Example: rotate


```doenet-editor-horiz
<slider name="angle" from="0" to="2pi"/>
<graph size="small">
  <circle  radius="7" filled/>
  <image rotate="$angle" layer="1" source='/media/bafkreifp2vbffgblocyvziejmp7hihzsx4dfkz6x4fayarcnfbb27pygaa.png' description='cartoon squirrel' asfilename='squirrel-clipart-md.png' mimeType='image/png' size="small" anchor="(1,2)" />
</graph>
```



By adjusting the `layer` attribute number, geometry can be placed in front of or behind other geometry. The default `layer` number is 0; all higher numbers are placed in front.
