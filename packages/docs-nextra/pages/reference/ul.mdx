# `<ul>{:dn}`

<ComponentDisplay name='ul'/>

`<ul>{:dn}` is a [Sectional](../document_structure/essentialConcepts#component-types)
component that renders an unordered (bulleted) list. The `<ul>{:dn}` component can only take [`<li>{:dn}`](li.mdx) 
components (list items) as children.

<AttrPropDisplay name='ul'/>

## Examples

### Example: `<ul>{:dn}` with nested `<li>{:dn}`


```doenet-editor-horiz
<p>Grocery list:</p>
<ul>
  <li>coffee</li>
  <li>cheese</li>
  <li>chocolate</li>
</ul>
```



`<ul>{:dn}` renders an unordered list, with list items given by `<li>{:dn}` children. `<ul>{:dn}` is a block component and should not contain any components other than `<li>{:dn}` children. Individual `<li>{:dn}` children may themselves contain graphs, problems, paragraphs, and other content.



---

### Example: `<ul>{:dn}` with more content


```doenet-editor-horiz
<ul>
  <li>
    <p><lorem generateSentences="4"/></p>
  </li>
  <li>
    <graph size="small" displayMode="inline"><shortDescription>An empty graph</shortDescription></graph>
  </li>
  <li>
    <problem>
      <p><answer><label>What is <m>\sqrt{16}</m></label>4</answer></p>
    </problem>
  </li>
</ul>
```



`<ul>{:dn}` components can only take `<li>{:dn}` children. Individual `<li>{:dn}` children may themselves contain graphs, problems, paragraphs, and other content.
