# ` <variance>{:dn}`


` <variance>{:dn}` is a [Math Operator](../document_structure/essentialConcepts/#componentTypes) 
component that renders the variance of a list of enclosed data values according to the equation: 


$$\mathbf{Var}(x) = \frac{\sum_{i=1}^n (x_{i} - \bar x)^2}{n-1}$$


In this equation, $\bar x$ represents 
the [` <mean>{:dn}`](mean.mdx) of the sample and $n$ the number of data points.  By default, the sample variance is 
calculated.  If instead, the `population` attribute is specified as "true", the equation 
becomes: 


$$\mathbf{Var}(x) = \frac{\sum_{i=1}^n (x_{i} - \bar x)^2}{n}$$

### Example: ` <variance>{:dn}` of a nested list of numbers


```doenet-editor-horiz
<variance>1 2 3 4 -1</variance>    
```



The ` <variance>{:dn}` component renders the variance of a list of numbers separated by spaces.



---

### Example: ` <variance>{:dn}` of a named list


```doenet-editor-horiz
<numberList name="numList">
  3 5 4 5 4 6
</numberList>

<p><m>\mathbf{Var}(x) = <variance>$numList</variance></m></p>  
```



The `<variance>{:dn}` component renders the variance of a named ` <numberList>{:dn}`.



---

### Example: ` <variance>{:dn}` of a user-input list


```doenet-editor-horiz
<p>Enter a list of numbers separated by commas: 
<mathInput name="mi"/>
</p>

<p>You've entered:   
  <numberList name="userList">$mi</numberList>
</p>

<p>The variance of your list of numbers is = 
  <variance>$userList</variance>
</p>  
```


Since there is no numberInput component, the ` <mathInput>{:dn}` is used to store a user's numerical list. This 
must then be defined as a ` <numberList>{:dn}`. The named ` <numberList>{:dn}` is then referenced by 
the ` <variance>{:dn}` component to compute the numerical variance.
