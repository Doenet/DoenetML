# ` <statement>{:dn}`

A ` <statement>{:dn}` is a sectional component that does not have auto-numbering.

## Attributes and Properties 
 <AttrDisplay name='statement'/> 
 <PropDisplay name='statement'/>

---

### Example: 


```doenet-editor-horiz
<section>
<p><lorem generateSentences="2"/></p>
</section>
<statement>
  <p>This is a statement.
  <lorem generateSentences="2"/>
  </p>
</statement>
<aside>
  <p><lorem generateSentences="2"/></p>
</aside>
```



This Sectional Component does not currently provide distinct styling.
