# `<solution>{:dn}`



<ComponentDisplay name='solution'/>

`<solution>{:dn}` is a [Sectional](../document_structure/essentialConcepts#component-types) component whose function is to display an explanation or problem solution.

<AttrPropDisplay name='solution'/>

## Examples

### Example: Default `<solution>{:dn}`


```doenet-editor-horiz
<problem>Move the point.
  <graph size="small">
    <shortDescription>A graph with a movable point</shortDescription>
    <point name="p"/>
  </graph>
  <answer>
    <award><when>$p != (0,0)</when></award>
  </answer>
  <solution>
    A nice explanation of all the fine details.
    <graph size="small"><shortDescription>example graph</shortDescription></graph>
  </solution>
</problem>
```



A `<solution>{:dn}` is placed within a `<problem>{:dn}`. Authors can control the conditions under which solutions may be viewed by students in the document-wide settings.
