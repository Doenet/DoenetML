# ` <coords>{:dn}`

` <coords>{:dn}` is a math component that renders a set of coordinates.

## Attributes and Properties 
 <AttrDisplay name='coords'/> 
 <PropDisplay name='coords'/>

---

### Example: ` <coords>{:dn}` rendered as a ` <math>{:dn}`


```doenet-editor-horiz
<coords>(2, 3)</coords>
<coords>(2, 3, 4)</coords>
```



The ` <coords>{:dn}` component renders as an ordered pair or an ordered triplet.



---

### Example: ` <coords>{:dn}` referenced by a ` <point>{:dn}`


```doenet-editor-horiz
<coords name="pXY">(2, 3)</coords>
<graph size="small">
  <point coords="$pXY"/>
</graph>
```



A named ` <coords>{:dn}` component is referenced in the `coords` attribute of a ` <point>{:dn}`.
