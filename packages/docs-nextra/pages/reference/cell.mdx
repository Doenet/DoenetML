# ` <cell>{:dn}`


` <cell>{:dn}` is a [Sectional](../document_structure/essentialConcepts#component-types)
component. It is the smallest element of a ` <tabular>{:dn}` block of text or 
data inside a ` <table>{:dn}`.

## Attributes and Properties 
 <AttrDisplay name='cell'/> 
 <PropDisplay name='cell'/>


---

### Example: ` <cell>{:dn}` inside a ` <table>{:dn}`


```doenet-editor-horiz
 <table>
    <tabular top="minor">
      <row>
        <cell>A</cell>
        <cell>B</cell>
        <cell>C</cell>
      </row>
      <row>
        <cell>游릮</cell>
        <cell>游댮</cell>
        <cell>游릭</cell>
      </row>
    </tabular>
  </table> 
```



Each ` <cell>{:dn}` component defines a single entry in the ` <table>{:dn}`.



---

### Attribute Example: colSpan


```doenet-editor-horiz
<table>
    <tabular top="minor" bottom="minor" left="minor" right="minor">
      <row>
        <cell>A</cell>
        <cell>B</cell>
        <cell>C</cell>
      </row>
      <row>
        <cell colSpan="2">游릮</cell>
        <cell>游릭</cell>
      </row>
      <row>
        <cell>1</cell>
        <cell colSpan="2">2</cell>
      </row>
    </tabular>
  </table> 
```



The `colSpan` attribute allows a single ` <cell>{:dn}` to span across 
multiple columns. This attribute can also be used across all rows to create 
visually wider "columns" within a table.



---

### Attribute Example: hAlign


```doenet-editor-horiz
 <table><title><attr>hAlign</attr> examples</title>
    <tabular top="minor" bottom="minor" left="minor" right="minor">
      <row>
        <cell>left</cell>
        <cell>center</cell>
        <cell>justify</cell>
        <cell>right</cell>
      </row>
      <row>
        <cell hAlign="left">游릮</cell>
        <cell hAlign="center">游댮</cell>
        <cell hAlign="justify">游릭</cell>
        <cell hAlign="right">游댯</cell>
      </row>
    </tabular>
  </table>    
```



The `hAlign` attribute governs the horizontal alignment of 
content within the ` <cell>{:dn}`. The default horizontal alignment is `left`.

### Property Example: Attributes as properties


```doenet-editor-horiz
  <table>
    <tabular top="minor">
      <row>
        <cell name="c1" bottom="major" hAlign="center"><math>pi/4</math></cell>
        <cell name="c2" right="minor" prefill="A"/>
      </row>
    </tabular>
  </table> 
  <p>First cell, <c>colSpan</c> = $c1.colSpan</p>
  <p>First cell, <c>hAlign</c> = $c1.hAlign</p>
  <p>First cell, <c>bottom</c> = $c1.bottom</p>
  <p>Second cell, <c>right</c> = $c2.right</p>
  <p>Second cell, <c>prefill</c> = $c2.prefill</p>
```



All of the attributes available to the ` <cell>{:dn}` component can also be accessed as properties.



---

### Property Example: inHeader


```doenet-editor-horiz
  <table>
    <tabular>
      <row header>
        <cell name="c1">A</cell>
        <cell>B</cell>
      </row>
      <row>
        <cell name="c3">游릮</cell>
        <cell>游댮</cell>
      </row>
    </tabular>
  </table>
  <p>Top row cell, <c>inHeader</c> = $c1.inHeader</p>
  <p>Bottom row cell, <c>inHeader</c> = $c3.inHeader</p>      
```



The `inHeader` property returns the boolean value corresponding to 
whether the ` <cell>{:dn}` occurs within a header ` <row>{:dn}`.
