# `<rsq/>{:dn}`

<ComponentDisplay name='rsq'/>

`<rsq/>{:dn}` is a [Paragraph Markup](../document_structure/essentialConcepts#component-types)
component that renders a right single quotation mark (’).


<AttrPropDisplay name='rsq'/>


## Examples

### Example: `<rsq/>{:dn}` in a paragraph


```doenet-example
<p>She replied, <lsq/>maybe<rsq/>, but it was hard to tell.</p>
```



For most cases, the [`<sq>{:dn}`](sq) component is a more convenient way to wrap text in
matching curly single quotation marks. Use [`<lsq/>{:dn}`](lsq) and `<rsq/>{:dn}` when you
need to place each mark independently.
