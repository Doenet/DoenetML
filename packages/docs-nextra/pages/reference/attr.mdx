# ` <attr>{:dn}`


` <attr>{:dn}` is a  [Paragraph Markup](../document_structure/essentialConcepts#component-types) 
component which renders the enclosed 
text in the default font for component attributes.

## Attributes and Properties 
` <attr>{:dn}` does not have any attributes or properties.



---


```doenet-example
<p>You can use the <attr>center</attr> attribute of the <tag>rectangle</tag> 
component to position it on the <tag>graph</tag>.</p>
```
