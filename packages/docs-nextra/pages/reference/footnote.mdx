# ` <footnote>{:dn}`

` <footnote>{:dn}` renders a numbered footnote above and to the right of the content it is adjacent to.

## Attributes and Properties 
 <AttrDisplay name='footnote'/> 
 <PropDisplay name='footnote'/>

---

### Example: Default ` <footnote>{:dn}`


```doenet-editor-horiz
<p>The Loch Ness monster<footnote>some more credible information about the monster</footnote></p>
```



A ` <footnote>{:dn}` is placed beside related content.



---

### Example: text


```doenet-editor-horiz
<p>The Abominable Snowman<footnote name="foot1">some more credible information about the monster</footnote></p>
<p><c>text</c> = $foot1.text</p>
```



The  `text` property renders the text value of the named ` <footnote>{:dn}`.



---

### Example: footnoteTag


```doenet-editor-horiz
<p>The Mothman<footnote name="foot1">some more credible information about the monster</footnote></p>
<p><c>footnoteTag</c> = $foot1.footnoteTag</p>
```



The  `footnoteTag` property renders the index associated with the named ` <footnote>{:dn}`.
