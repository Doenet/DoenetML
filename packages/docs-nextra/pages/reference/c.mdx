# `<c>{:dn}`

<ComponentDisplay name='c'/>

`<c>{:dn}` is a [Paragraph Markup](../document_structure/essentialConcepts#component-types) component 
which renders the enclosed text in the default font for illustrating code examples.

<AttrPropDisplay name='c'/>

## Examples

### Example: `<c>{:dn}` used around code snippet


```doenet-example
<p>The value of the <tag>boolean</tag> is <c>false</c>.</p>
```



Use the `<c>{:dn}` component to enclose text that represents a code snippet.
