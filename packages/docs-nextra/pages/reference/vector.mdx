# ` <vector>{:dn}`

` <vector>{:dn}` renders a vector when nested within a [` <graph>{:dn}`](graph.mdx). Outside a graph, it can be used in vector computations.

## Attributes and Properties 
 <AttrDisplay name='vector'/> 
 <PropDisplay name='vector'/>


---

### Example: Default ` <vector>{:dn}`


```doenet-editor-horiz
<graph size="small" xmin="-2" xmax="2" ymin="-2" ymax="2">
  <vector/>
</graph>
```



A default ` <vector>{:dn}` without any additional attributes has `tail` at the origin and `head` at $(1,0)$.



---

### Example: ` <vector>{:dn}` math


```doenet-editor-horiz
<p><m>a = </m><vector name="a">(1,2)</vector></p>
<p><m>b = </m><vector name="b">(3,4)</vector></p>
<p><m>a + b = 
  <math simplify>$a + $b</math></m></p>
<p><m>a - b = 
  <math simplify>$a - $b</math></m></p>
<p><m>c \ \bf{a} = </m> 
  <math simplify>c $a</math></p>
```



A ` <vector>{:dn}` within a ` <math>{:dn}` follows vector operations.



---

### Example: x,y,(z*)


```doenet-editor-horiz
<p>Drag the head and the tail of the vector, or the vector itself.</p>
<graph size="small" grid="1 1" xmin="-6" xmax="6" ymin="-6" ymax="6">
  <vector name="v1" x="3" y="4"/>
</graph>
<p><c>x</c> = $v1.x</p>
<p><c>y</c> = $v1.y</p>      
```



The `x` and `y` attributes allow the vector to be specified by its coordinates or row elements for a vector given by $\bf{v}$$ = (x, y, z)$.



---

### Example: xs


```doenet-editor-horiz
<p>Drag the head and the tail of the vector, or the vector itself.</p>
<graph size="small" grid="1 1" xmin="-6" xmax="6" ymin="-6" ymax="6">
  <vector name="v1" xs="3 4" tail="(1,1)"/>
</graph>
<p><c>displacement</c> = $v1.displacement</p>
<p><c>displacement[2]</c> = $v1.displacement[2]</p>  
```



The `xs` attribute allows the vector to be specified by its displacements in list format. To access or render the list information, use the `displacement` property.



---

### Example: displacement


```doenet-editor-horiz
<p>Drag the head and the tail of the vector, or the vector itself.</p>
<graph size="small" grid="1 1" xmin="-6" xmax="6" ymin="-6" ymax="6">
  <vector name="v1" displacement="(3,4)" tail="(1,1)"/>
</graph>
<p><c>displacement</c> = $v1.displacement</p>
<p><c>displacement[2]</c> = $v1.displacement[2]</p>  
```



Use the `displacement` property to specify the displacement of the vector in each direction from its tail position.



---

### Example: head, tail


```doenet-editor-horiz
<p>Drag the head and the tail of the vector, or the vector itself.</p>
<graph size="small" grid="1 1" xmin="-6" xmax="6" ymin="-6" ymax="6">
  <vector name="v1" head="(4,5)" tail="(1,1)"/>
  $v1.head{styleNumber="4"}
  $v1.tail{styleNumber="2"}
</graph>
<p><c>head</c> = $v1.head</p>
<p><c>head[2]</c> = $v1.head[2]</p>  
<p><c>tail</c> = $v1.tail</p>
<p><c>tail[2]</c> = $v1.tail[2]</p>   
```



Use `head` and `tail` to specify the head and tail coordinates of the vector directly.

### Example: displayWithAngleBrackets



If this is the desired rendering format, use the attribute `displayWithAngleBrackets`.

### Example: draggable


```doenet-editor-horiz
<p>Try to drag the head and the tail of each vector, or the vectors themselves.</p>
<graph size="small" grid="1 1" xmin="-6" xmax="6" ymin="-6" ymax="6">
  <vector name="v1" tail="(-2,0)">(3,4)
    <label>Draggable</label>
  </vector>
  <vector name="v2" styleNumber="2" draggable="false" tail="(-2,-3)">(3,4)
    <label>Not draggable</label>
  </vector>
</graph>
<p><c>v1.draggable</c> = $v1.draggable</p>
<p><c>v2.draggable</c> = $v2.draggable</p>
```



The `draggable` attribute determines whether the ` <point>{:dn}` can be dragged around the ` <graph>{:dn}`. All graphical objects are draggable by default.



---

### Example: headDraggable, tailDraggable


```doenet-editor-horiz
<booleanInput name="hD">
  <label>Head draggable</label>
</booleanInput>
<booleanInput name="tD">
  <label>Tail draggable</label>
</booleanInput>
<graph size="small" grid="1 1" xmin="-6" xmax="6" ymin="-6" ymax="6">
  <vector name="v1" headDraggable="$hD" tailDraggable="$tD">(3,4)</vector>
</graph>
<p><c>headDraggable</c> = $v1.headDraggable</p>
<p><c>tailDraggable</c> = $v1.tailDraggable</p>
```



The head and tail of a ` <vector>{:dn}` are draggable by default. Use the `headDraggable` or `tailDraggable` attribute set to false to disable users from changing the direction and length of the ` <vector>{:dn}`. The vector as a whole can still be repositioned on the ` <graph>{:dn}`.
