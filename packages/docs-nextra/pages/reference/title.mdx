# ` <title>{:dn}`


` <title>{:dn}` is a [Sectional](../document_structure/essentialConcepts/#componentTypes)
component that renders a title with default formatting to match the heierarchy of its 
associated sectional component. A ` <title>{:dn}` placed at the top of the document 
creates a document title.

## Attributes and Properties 
 The ` <title>{:dn}` component has no attributes.
 <PropDisplay name='title'/>

---

### Example: ` <title>{:dn}` for a ` <section>{:dn}`


```doenet-editor-horiz
<section>
  <p>This section has a default title.</p>
</section>

<section><title>How to change a lightbulb</title>
  <p>This section has a custom title.</p>
</section>
```



A ` <title>{:dn}` placed inside a ` <section>{:dn}` overrides the default title for that ` <section>{:dn}`.



---

### Example: ` <title>{:dn}` for a subsection


```doenet-editor-horiz
<section><title>Really important</title>
  <p>This is the larger section.</p>
  <section><title>Slightly less important</title>
    <p>This section is nested inside and creates a subsection.</p>
  </section>
</section>
```



A ` <section>{:dn}` nested inside another ` <section>{:dn}` creates a subsection of lower hierarchy.
