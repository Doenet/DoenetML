# ` <styleDefinitions>{:dn}`

The ` <styleDefinitions>{:dn}` component is a container element for one or more ` <styleDefinition>{:dn}` components. It must be placed in a ` <setup>{:dn}` block.

## Attributes and Properties 
 <AttrDisplay name='styleDefinitions'/> 
 <PropDisplay name='styleDefinitions'/>

### Example: Defining custom styles in a ` <styleDefinitions>{:dn}` block


```doenet-editor-horiz

<styleDefinitions>
  <styleDefinition styleNumber="10" 
    fillColor="green" 
    fillOpacity="0.5" 
    markerSize="6" 
    markerStyle="triangle" 
    markerColor="green" 
    lineColor="green" 
    lineOpacity="0.7"  
    lineWidth="2"
    lineStyle="dashed"/>
  <styleDefinition styleNumber="11" 
    fillColor="hotpink" 
    fillOpacity="0.3" 
    markerSize="6" 
    markerStyle="diamond" 
    markerColor="hotpink" 
    lineColor="hotpink" 
    lineOpacity="0.7"  
    lineWidth="4"
    lineStyle="dotted"/>
</styleDefinitions>      

<graph>
  <line name="greenLine" styleNumber="10">y = x</line>
  <line name="pinkLine" styleNumber="11">y = x + 2</line>
  <line name="defLine">y = -x</line>
  <point name="greenPt" styleNumber="10">(3,3)</point>
  <point name="pinkPt" styleNumber="11">(3,5)</point>
  <point name="defPt">(-3,3)</point>
  <circle name="greenCircle" styleNumber="10" center="(0,0)" filled/>
  <circle name="pinkCircle" styleNumber="11" center="(-6,-4)" filled/>
  <circle name="defCircle" filled center="(3,-3)"/>
</graph>
```



In this example, two custom styles have been created within a ` <styleDefinitions>{:dn}` block. These styles are assigned to the `styleNumber`s $10$ and $11$, which are used in graphing geometry.


Note that the ` <styleDefinitions>{:dn}` block occurs within a ` <setup>{:dn}` block, and can be located anywhere in the document.
