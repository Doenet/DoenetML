# ` <question>{:dn}`

` <question>{:dn}` is a [Sectional](../document_structure/essentialConcepts#component-types)
component that renders an auto-numbered block of content. Inline components such as paragraphs 
(`<p>{:dn}`) and other components can be nested within an ` <question>{:dn}` to group and order 
content. Some examples are: another ` <question>{:dn}`, a ` <section>{:dn}`, 
a ` <figure>{:dn}`, or an ` <example>{:dn}`.


` <question>{:dn}` is an alias for ` <problem>{:dn}`, and behaves identically.
Please see the [`<problem>{:dn}`](problem) documentation for examples.
