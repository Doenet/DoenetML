# ` <product>{:dn}`

` <product>{:dn}` is a [Math Operator](../document_structure/essentialConcepts#component-types)
component that renders the product of its arguments.

## Attributes and Properties 
 <AttrDisplay name='product'/> 
 <PropDisplay name='product'/>

---

### Example: ` <product>{:dn}` of a nested list of numbers


```doenet-editor-horiz
<product>1.2 2.5 1 4.1 3</product>
```



` <product>{:dn}` renders the product of values in a list of numbers separated by spaces.



---

### Example: ` <product>{:dn}` of a named list


```doenet-editor-horiz
<numberList name="numList">
  1.2 9.9 17.3
</numberList>

<p>product = <product>$numList</product></p> 
```



` <product>{:dn}` renders the product of values in a named ` <numberList>{:dn}`.



---

### Example: ` <product>{:dn}` of a randomly selected set of numbers


```doenet-editor-horiz
<selectRandomNumbers 
  numToSelect="4" 
  from="-10" 
  to="10" 
  name="randomNumberList"
/>

<p>The list of random numbers selected is:
  $randomNumberList
</p>

<p>product of selected numbers: <product>$randomNumberList</product></p>
```



` <product>{:dn}` renders the product of values in a list of numbers that are generated 
with the ` <selectRandomNumbers/>{:dn}` component. 



---

### Example: ` <product>{:dn}` of symbolic maths


```doenet-editor-horiz
<mathList name="mList">
  x 3y 2x^2 y -3x x^2
</mathList>
<p>product = 
  <product name="productOfMaths">$mList</product>
</p> 

<p>Simplified = 
  <math extend="$productOfMaths" simplify/>
</p> 
```



` <product>{:dn}` renders the product of values in a named ` <mathList>{:dn}`. Since ` <product>{:dn}` is a math component, ` <math>{:dn}` attributes like `simplify` are also available.
