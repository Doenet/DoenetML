# ` <introduction>{:dn}`



` <introduction>{:dn}` is a [Sectional](../document_structure/essentialConcepts#component-types)
component that is analogous to a `<p>{:dn}` when rendered on Doenet.org. In this context, it does not have 
any distinctive UI features. It does not have a title and it is not auto-numbered.


## Attributes and Properties 
The ` <introduction>{:dn}` does not have attributes.
 <PropDisplay name='introduction'/>



---

### Example: ` <introduction>{:dn}` basic use


```doenet-example
<introduction>
  <p>In this chapter we will cover several important topics:</p>
  <ul>
    <li>topic 1</li>
    <li>topic 2</li>
    <li>topic 3</li>
  </ul>
</introduction> 

<section>
  <lorem generateWords="20"/>
</section>
```

The ` <introduction>{:dn}` component is analagous to the `<p>{:dn}`
componenet on Doenet.org.
