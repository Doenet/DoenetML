# `<alert>{:dn}`

## What it does
`<alert>{:dn}` is a [**Paragraph Markup**](../componentClasses) component which renders the enclosed text in bold font.

## Attributes and Properties
The `<alert>{:dn}` component has no attributes.

<PropDisplay name='alert'/>

## Examples

### Example: Basic use

The text that should be bold is placed within the opening and closing tags of the `<alert>{:dn}` component.
```doenet-editor-horiz
<p>Take the <alert>right</alert> turn.</p>
```
