# `<nbsp/>{:dn}`

<ComponentDisplay name='nbsp'/>

`<nbsp/>{:dn}` is a [Paragraph Markup](../concepts/essentialConcepts#component-types) 
component that renders a non-breaking space in a paragraph.


<AttrPropDisplay name='nbsp'/>


## Examples

### Example: `<nbsp/>{:dn}` in a paragraph


```doenet-editor-horiz
<p>The time has come, the Walrus said, to talk of many things,</p>
<p>Of shoes,</p>
<p><nbsp/><nbsp/><nbsp/>and ships,</p>
<p><nbsp/><nbsp/><nbsp/><nbsp/><nbsp/>
    and ceiling wax...</p>
<p>Of carpenters,</p>
<p><nbsp/><nbsp/><nbsp/><nbsp/><nbsp/>
    and kings.</p>
```



The `<nbsp/>{:dn}` component is used to force additional white space where desired in a paragraph.
