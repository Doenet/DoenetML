# `<lsq/>{:dn}`

<ComponentDisplay name='lsq'/>

`<lsq/>{:dn}` is a [Paragraph Markup](../concepts/essentialConcepts#component-types)
component.


<AttrPropDisplay name='lsq'/>


## Examples

### Example: `<lsq/>{:dn}` in a paragraph


```doenet-example
<p>She replied, <lsq/>maybe<rsq/>, but it was hard to tell.</p>
```



For most cases, the [`<sq>{:dn}`](sq) component is a more convenient way to wrap text in
matching curly single quotation marks. Use `<lsq/>{:dn}` and [`<rsq/>{:dn}`](rsq) when you
need to place each mark independently.
