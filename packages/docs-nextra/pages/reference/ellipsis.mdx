# `<ellipsis>{:dn}`

<ComponentDisplay name='ellipsis'/>

`<ellipsis/>{:dn}` is a [Paragraph Markup](../document_structure/essentialConcepts#component-types) 
component consisting of three closely spaced dots.

<AttrPropDisplay name='ellipsis'/>


## Examples

### Example: <tage>ellipsis</tage> in a `<p>{:dn}`


```doenet-example
<p>I'm thinking <ellipsis/></p>
```



An `<ellipsis/>{:dn}` is rendered in a paragraph.
