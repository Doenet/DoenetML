# ` <ellipsis>{:dn}`


` <ellipsis/>{:dn}` renders an ellipsis symbol consisting of three closely spaced dots.



---

### Example: <tage>ellipsis</tage> in a ` <p>{:dn}`


```doenet-editor-horiz
<p>I'm thinking <ellipsis/></p>
```



An ` <ellipsis/>{:dn}` is rendered in a paragraph.
