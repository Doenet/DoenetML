# `<selectRandomNumbers/>{:dn}`

` <selectRandomNumbers/>{:dn}` creates named variants from a generates random numbers from a specified range of values or within a normal distribution around a specified mean.

## Attributes and Properties 
 <AttrDisplay name='selectRandomNumbers'/> 
 <PropDisplay name='selectRandomNumbers'/>

---

### Example: Default `<selectRandomNumbers/>{:dn}`


```doenet-editor-horiz

<selectRandomNumbers name="selection" />

<p>Number selected = $selection</p>
```



Without any attributes specified, the default selection  `type` is `uniform`, and the default interval is $[0, 1)$. Select a different page variant to view a different selection.



---

### Example: numToSelect


```doenet-editor-horiz
<p>
  <asList>
    <selectRandomNumbers numToSelect="4" name="randomNumbers"/>
  </asList>
</p>
```



The ` <selectRandomNumbers/>{:dn}` component is used with the `numToSelect` attribute to select multiple random values in the default interval $[0,1)$. The default `type` is `uniform`. The component is nested within an [` <asList>{:dn}`](asList.mdx) component for greater clarity when rendering.



---

### Example: assignNames


```doenet-editor-horiz

<selectRandomNumbers assignNames="a b c" numToSelect="3"/>
<selectRandomNumbers name="d"/>

<p>a = $a</p>
<p>b = $b</p>
<p>c = $c</p>
<p>d = $d</p>
```



The `assignNames` attribute is used to specify unique identifiers for each selection where one more than one selection is required. If only one selection is required, the `name` of the selection itself assumes the selected value.


 Select a different page variant to view a different selection.



---

### Example: from, to


```doenet-editor-horiz
<p>
<asList>
  <selectRandomNumbers name="x" from="-5" to="5" numToSelect="5" />
</asList>
</p>
```



The ` <selectRandomNumbers/>{:dn}` component is used with the `from` and the `to` attributes to modify the default selection interval. The default `type` is `uniform`.


 Select a different page variant to view a different selection.



---

### Example: type = "discreteUniform"


```doenet-editor-horiz
<p><asList>
    <selectRandomNumbers name="x" type="discreteUniform" from="0" to="10" numToSelect="5"/>
</asList></p>
```



The `type` attribute of the ` <selectRandomNumbers/>{:dn}` component is specified as `discreteUniform`. The `discreteUniform` selection generates values from an evenly spaced range defined by the attributes `to`, `from`, and `step`. If left unspecified, the default `step` is 1.


 Select a different page variant to view a different selection.



---

### Example: exclude


```doenet-editor-horiz

  <selectRandomNumbers name="x" assignNames="a b c" type="discreteUniform" from="-4" to="4" numToSelect="3" exclude="0"/>

<p><math simplify="numberspreserveorder">$a x^2 + $b x + $c</math></p>
```



The `type` attribute of the ` <selectRandomNumbers/>{:dn}` component is specified as `discreteUniform`. Values can be excluded from the specified range used with this type using the `exclude` attribute.


 Select a different page variant to view a different selection.

### Example: type = "gaussian"


```doenet-editor-horiz
<p>
  <asList>
    <selectRandomNumbers name="x" type="gaussian" numToSelect="10" displayDigits="4"/>
  </asList>
</p>      
```



The `type` attribute of the ` <selectRandomNumbers/>{:dn}` component is specified as `gaussian`. The `gaussian` selection generates standard normal variables (mean 0, variance 1). Alternatively, the `mean` as well as the `variance` or the `standardDeviation` attributes may be defined.


 Select a different page variant to view a different selection.
