# Introduction



<Callout type="info">
## DoenetML is...

* a semantic markup language used for the rapid generation of interactive webpages

* especially powerful for creating dynamic educational content in STEM courses

* easy to learn, even for authors with no programming experience
</Callout>

---


## Explore three different learning options:
1. [**Tutorials**](tutorials/quickStart.mdx) : work through guided example projects in video and/or text format
2. [**Reference**](reference/componentIndex.mdx) : technical documentation and component examples
3. [**Sandbox**](sandbox0.mdx) : coming soon


 DOENET is a core technology of the [PROSE](https://prose.runestone.academy/) ecosystem for open educational resources.
