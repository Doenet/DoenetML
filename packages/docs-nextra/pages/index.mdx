# Introduction

## What is DoenetML?

* **DoenetML** is a semantic markup language used for the rapid generation of interactive webpages

* **DoenetML** is especially powerful for creating dynamic educational content in STEM courses

* **DoenetML** is easy to learn, even for authors with no programming experience

---

## Which path will you take?


Choose from one of three different learning paths:
1. Read the Quick Start page and work through our guided project tutorials in the **Learn** section
2. Dig deeper into the technicalities of the language and advanced topics in the **References** section
3. Immediately tinker around with DoenetML code snippets in the **Sandbox** section


 DOENET is a core technology of the [PROSE](https://prose.runestone.academy/) ecosystem for open educational resources.
