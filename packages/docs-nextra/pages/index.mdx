# What is DoenetML?

<Callout type="info">
* a semantic markup language used for the rapid generation of interactive webpages

* especially powerful for creating dynamic educational content in STEM courses

* easy to learn, even for authors with no programming experience
</Callout>

---


## Explore three different learning options:
1. [**Tutorials**](tutorials/introdTutorial/iT1-1-writing-mathematics.mdx) : work through guided example projects in video and/or text format
2. [**Reference**](reference/essentialConcepts.mdx) : technical documentation and component examples
