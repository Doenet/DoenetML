# What is DoenetML?

<Callout type="info">
* a semantic markup language used for the rapid generation of interactive webpages

* especially powerful for creating dynamic educational content in STEM courses

* easy to learn, even for authors with no programming experience
</Callout>

---


## How to use this document:
1. [**Components**](Components/componentIndex.mdx): the language of DoenetML is built from components. 
    On every page, a list of components will always appear in the navigation panel on the left.
    Beginners are encouraged to check out the [Alphabetical Index](Components/componentIndex.mdx) first.
2. [**Tutorials**](tutorials/introdTutorial/iT1-1-writing-mathematics.mdx) : work through guided example projects
3. [**Reference**](reference/essentialConcepts.mdx) : technical documentation regarding document layout and structure
4. [**Sandbox**](sandbox/sandbox0.mdx) : practice bite-size coding challenges 


## How to use DoenetML:
1. [**Share on Doenet.org**](tutorials/shareOnDoenet.mdx): Collaborate with your colleagues, test new activities using Quick Assign, or contribute
    your own work to the growing library of peer-reviewed problems on [the brand new Doenet Website](https://alpha.doenet.org/)
2. [**Embed in a PreTeXt book**](tutorials/embedInPretext.mdx)
3. [**Assign in Runestone**](tutorials/runestone.mdx)
3. [**Embed in a Google Site**](tutorials/googleSite.mdx)
3. [**Assign in Canvas**](tutorials/canvas.mdx)

 DOENET is a core technology of the [PROSE](https://prose.runestone.academy/) ecosystem for open educational resources.
